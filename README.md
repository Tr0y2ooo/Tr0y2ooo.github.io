# Tr0y2ooo.github.io
This is a page for training engineering-math

The homepage allows users to select different units of engineering mathematics. Upon entering the chosen unit, users can view notes. After reviewing the notes, they can click the bottom-left button to return to the homepage or switch to the problems page. Similarly, on the problems page, users can click the bottom-left button to return to the homepage or switch to the notes page. Additionally, there is a button for randomly selecting problems that users can utilize.

[Link here](https://tr0y2ooo.github.io/)

AI4Kid 認識人工智慧

認識人工智慧

  • 人工智慧
    • 計算機模仿人類思考進而模擬人類的能力/行為。
  • 機器學習
    • 從資料中學習模型。
  • 深度學習
    • 利用多層的非線性學習資料表徵。

數據->演算法->預測

機器學習與醫療應用

  • 機器學習的五大流程
    • 定義問題->蒐集資料->處理資料集->訓練模型->推論與預測

定義問題

  • 判斷問題適不適合用機器學習來解決

蒐集資料

  • 告訴電腦資料的標籤是什麼
  • 資料越全面，機器學習的辨識率越高

處理資料集

  • 找出各個變數之間的交互關係

訓練模型

  • 挑選適合的演算法
  • 可以同時使用多個演算法
  • 通常會將資料分成訓練資料集和測試資料集(80%/20%)


推論與預測

  • 會給出一個最有可能的答案
  • 不夠好可以回到蒐集資料重新訓練一次

機器學習迴歸分析之醫療應用

  • 分類 - 判別有限類別中的某一類別
  • 迴歸 - 估算一個預測值

監督式學習

  • 資料已有標記，運用已標記資料做訓練

非監督式學習

  • 沒有答案，只有輸入特徵期望機器找出樣本之間隱含關係

強化式學習

  • 可能手上沒有任何資料，直接讓模型執行，再將執行結果反饋回去做訓練

線性迴歸

  • 將資料點放在直角坐標圖上，找到一條和各個點距離(誤差總和)最小的線

專有名詞

CNN

CNN是一種深度學習模型，主要用於處理和分析圖像數據。它受到生物視覺系統的啟發，能夠自動從圖像中學習特徵並做出預測。

Convolutional Layer

卷積層是CNN的核心組件之一。它使用卷積操作來檢測圖像中的特徵，例如邊緣、紋理和形狀。通過選擇不同的卷積核（類似於過濾器），卷積層可以學習不同類型的特徵。

舉例來說，如果我們要訓練一個CNN來辨識貓和狗的圖像，卷積層的作用就是檢測圖像中的一些基本特徵，例如邊緣、眼睛、鼻子等。通過這些特徵的組合，CNN可以辨識出圖像中的貓或狗。

Pooling Layer

池化層通常跟在卷積層後面。它的作用是減少特徵圖的尺寸，同時保留重要的特徵。常見的池化操作是取區域內的最大值（稱為最大池化），這樣可以縮小圖像的空間尺寸。

舉例來說，如果我們使用最大池化將一張圖像的尺寸縮小為原來的一半，那麼池化層就會選擇每個區域中的最大值作為該區域的特徵。這樣可以減少計算量，同時保留圖像中最重要的特徵。

Fully Connected Layer

全連接層通常在卷積層和最後的輸出層之間。它的作用是將卷積層和池化層提取的特徵進行展平，並將其輸入到一個全連接神經網絡中，以進行最終的分類或預測。

舉例來說，如果我們的CNN要辨識圖像中的貓或狗，全連接層會將卷積層和池化層提取的特徵展平為一個長向量，然後通過全連接層進行分類，例如判斷是貓還是狗。

Activation Function

激活函數是在神經網絡中引入非線性性質的函數。在CNN中，激活函數通常被應用在每個神經元的輸出上，以增加模型的表達能力。

舉例來說，常用的激活函數之一是ReLU（整流線性單元），它將負數輸入設為零，並保留正數輸入。這樣可以引入非線性，使得神經網絡能夠學習更複雜的特徵和模式。

Data Augmentation

數據增強是一種技術，通過對訓練數據進行隨機變換和增強，來增加數據的多樣性。這樣可以提高模型的泛化能力，並減輕過擬合的問題。

舉例來說，對於圖像分類任務，數據增強技術可以通過隨機縮放、旋轉、平移、翻轉等操作來生成更多的訓練圖像。這樣可以讓模型在看到不同角度、大小和位置的圖像時都能夠有良好的表現。

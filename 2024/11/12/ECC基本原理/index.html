ECC

橢圓曲線密碼學(Elliptic Curve Cryptography)

ECC的主要優勢是它相比RSA加密演算法使用較小的密鑰長度並提供相當等級的安全性
• RSA : 1024 bit | 15360 bit
• ECC : 160 bit | 512 bit

函式為：y^2 = x^3 + (a)x + (b) 根據參數a,b的不同會有不同形狀

shape

加密方法

任意選擇兩個點 P, Q畫⼀直線，將會通過第三個點 R

• 我們將此定義為 P + Q + R = 0

剛剛定義的 P + Q + R = 0 推出 —->> P + Q = -R

實作

加上mod可以增加有限性，離散化

當兩個點都在同一個點 –> P = Q 的情況下

斜率

p + p = 2p

這邊的加法非一般四則運算之加法，內容經過一些演算法

p加到最後會循環，該循環數稱之為Cardinarity
